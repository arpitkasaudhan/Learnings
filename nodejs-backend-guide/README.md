## ğŸ¯ Real-World API Example

By the end of this guide, you'll build APIs like this:

### Authentication APIs
```javascript
POST   /api/auth/register      - Register new user
POST   /api/auth/login         - Login user
POST   /api/auth/logout        - Logout user
POST   /api/auth/refresh       - Refresh token
POST   /api/auth/forgot-password - Send reset email
POST   /api/auth/reset-password  - Reset password
GET    /api/auth/me            - Get current user
```

### Car Listing APIs
```javascript
GET    /api/cars               - Get all cars (with filters, pagination)
GET    /api/cars/:id           - Get single car
POST   /api/cars               - Create new listing (protected)
PUT    /api/cars/:id           - Update listing (protected, owner only)
DELETE /api/cars/:id           - Delete listing (protected, owner only)
POST   /api/cars/:id/images    - Upload car images (protected)
GET    /api/cars/search        - Search cars
GET    /api/cars/featured      - Get featured cars
```

### User APIs
```javascript
GET    /api/users/profile      - Get user profile (protected)
PUT    /api/users/profile      - Update profile (protected)
GET    /api/users/listings     - Get user's listings (protected)
GET    /api/users/favorites    - Get favorites (protected)
POST   /api/users/favorites/:carId - Add to favorites (protected)
DELETE /api/users/favorites/:carId - Remove from favorites (protected)
```

## ğŸ“– API Response Format

You'll learn to design consistent API responses:

### Success Response
```json
{
  "success": true,
  "message": "Car created successfully",
  "data": {
    "id": "507f1f77bcf86cd799439011",
    "brand": "Honda",
    "model": "City",
    "price": 1500000
  }
}
```

### Error Response
```json
{
  "success": false,
  "message": "Validation error",
  "errors": [
    {
      "field": "price",
      "message": "Price must be greater than 0"
    }
  ]
}
```

### Paginated Response
```json
{
  "success": true,
  "data": [...],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 100,
    "pages": 10
  }
}
```
