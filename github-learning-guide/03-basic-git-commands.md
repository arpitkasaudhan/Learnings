# Lesson 3: Basic Git Commands

## Essential Commands You'll Use Every Day

### Initialize Repository
```bash
git init
```

### Check Status
```bash
git status
```

### Add Files
```bash
git add <file>
git add .
git add -A
```

### Commit
```bash
git commit -m "message"
git commit -am "message"
```

### View History
```bash
git log
git log --oneline
git log --graph --oneline --all
```

### Push/Pull
```bash
git push
git pull
```
