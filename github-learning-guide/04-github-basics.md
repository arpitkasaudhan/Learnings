# Lesson 4: GitHub Basics

## Working with GitHub

### Clone Repository
```bash
git clone <url>
```

### Add Remote
```bash
git remote add origin <url>
git remote -v
```

### Push to GitHub
```bash
git push -u origin main
git push
```

### Pull from GitHub
```bash
git pull origin main
git pull
```

### Create Repository on GitHub
1. Go to github.com
2. Click "New repository"
3. Fill in details
4. Click "Create repository"
